Required Methods§

fn scene_tree(&self) -> &Gd<Node>

Get access to current scene tree.

Access is provided relatively to the GdTestRunner position in the current scene (it’s recommended that it is a root node of the scene).

Use it if you need to do something else besides getting access to some specific node, especially during benchmarking. For node retrieval you can use CaseContext::get_node and CaseContext::get_node_as.
